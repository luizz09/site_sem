Sistemas de Medições de Grandezas Físicas

O que é temperatura?

- Temperatura é uma grandeza física escalar que mede o grau da agitação ou energia cinética das moléculas e átomos que compõem um corpo ou sistema.

O que é calor?

- Calor é a transferência espontânea de energia térmica entre corpos ou sistemas com diferentes temperaturas. (fluindo sempre do mais quente para o mais frio até atingirem o equilíbrio térmico).

Como medir a temperatura?

- A temperatura, por padrão, é medida com um termômetro.

O que é medida?

- Medida é a quantificação de uma grandeza física (como comprimento, massa, tempo ou volume) por meio da comparação com um padrão pré-definido, chamado unidade de medida.

O que é precisão na medida?

- É quando as medidas estão próximas entre si, mas não necessariamente corretas.

Como exemplo: Valor Real: 10 | 9,5 / 9,6 / 9,7 -> estão bem próximas de si, ou seja, alta precisão mas baixa exatidão.

O que é exatidão de medida?

- É quando está perto do valor real, mas não necessariamente próximas entre si.

Como exemplo: Valor Real: 10 | 9,8 / 10,2 / 10,1 -> estão perto do valor real, ou seja, alta exatidão mas menor precisão.

Quais os tipos de sensores de temperatura analógicos?

- Termistor (NTC/PTC); Termopar; RTD (ex: PT100); Sensor LM35.

Quais os tipos de termopares?

- Os tipos de termopares mais comuns são o Tipo K (Níquel-Cromo / Níquel-Alumínio), que é o mais utilizado; o Tipo J (Ferro / Constantan); o Tipo T (Cobre / Constantan); o Tipo E (Níquel-Cromo / Constantan); e os Tipos S, R e B, que são mais utilizados em medições de temperaturas muito altas.

Como selecionar um sensor de temperatura?

- Olhar principalmente: Faixa de medição (escala); Precisão; Tempo de resposta; Ambiente (industrial, doméstico, etc).

Qual é a escala analógica do arduino (0 - 1023 => entrada e 0 - 255 => saída)

- Entrada: 0 a 1023 (10 bits); Saída PWM: 0 a 255 (8 bits).

O que é o arduíno?

- É uma placa eletrônica programável usada para automação, robótica e projetos eletrônicos.

O que é entrada analógica e o que é entrada digital no arduíno?

- Analógica: são valores contínuos 0-1023 (que podem ser usados para medir: temperatura, luz, etc); Digital: é apenas 0 ou 1 (ligado/desligado).

O que é entrada analógica e o que é entrada digital no CLP?

- Analógica: sinais contínuos (ex: 4–20 mA, 0–10V); Digital: sinais binários (liga/desliga).

O que é saída analógica (PWM) arduíno?

- PWM é um sinal digital que simula um valor analógico variando o tempo ligado/desligado. Exemplo: Controle de brilho de LED.

O que é saída analógica no CLP?

- São sinais contínuos (ex: 0–10V ou 4–20mA); Usado para controlar: Válvulas; Inversores de frequência.

O que é o CLP? (Modelo da CPU 1214c do tipo DC/DC/DC)

- O CLP é um equipamento eletrônico usado para automatizar processos, recebendo sinais de entrada, processando e acionando saídas. No modelo CPU 1214C DC/DC/DC, a alimentação, entradas e saídas são todas em corrente contínua (DC ou CC em português).

O que é a Ponte de Wheatstone?

- É um circuito usado para medir resistências com alta precisão, muito usado com sensores.

O que é multímetro?

- É um instrumento de medição que serve para medir grandezas elétricas: Tensão (V); Corrente (A); Resistência (Ω).

O que é medida de continuidade?

- É um teste para ver se há passagem de corrente em um circuito. Se apitar = circuito fechado.

O que é dimerização?

- É o controle da intensidade de luz, exemplo: Aumentar ou diminuir o brilho de um LED.
